Installation

Install via bench CLI or get from Frappe Cloud Marketplace.

Terminal
# Install Frappe WhatsApp (required)
$ bench get-app https://github.com/shridarpatil/frappe_whatsapp

# Install WhatsApp Chat ui (optional)
$ bench get-app https://github.com/shridarpatil/whatsapp_chat

# Install WhatsApp Chatbot (optional)
$ bench get-app https://github.com/shridarpatil/frappe_whatsapp_chatbot

# Install apps on your site
$ bench --site your-site.localhost install-app frappe_whatsapp
