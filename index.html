The suite

Three apps, one complete solution

Each app handles a specific aspect of WhatsApp integration. Use them together or pick what you need.

01

Frappe WhatsApp Core

The foundation. Connect your WhatsApp Business API and start sending messages, templates, and notifications from your Frappe/ERPNext system.

  • Multi-account support
  • Two-way messaging
  • Template management & sync from Meta
  • WhatsApp Flows (interactive forms)
  • Bulk messaging with variables
  • WhatsApp Notifications (DocType events)
  • Media support (images, docs, audio, video)

Whatomate

Not on Frappe? No problem. Get the same powerful WhatsApp automation as a standalone application.

Lightning Fast

Built with Go for blazing performance. Handle thousands of messages with minimal resources.

Single Binary

No dependencies, no runtime needed. Just download and run. Deploy anywhere in seconds.

Human Handoff

Smart escalation with message pool. Agents pick from pre-written responses for faster support.

Modern UI

Beautiful Vue.js interface. Manage conversations, templates, and automations with ease.

Any Stack

REST API for easy integration. Works with Node.js, Python, PHP, Ruby — whatever you use.

100% Free

Open source under MIT license. Use it, modify it, deploy it anywhere. No strings attached.
