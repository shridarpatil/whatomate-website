Quick start

Get started in minutes

Deploy with Docker Compose in four simple steps.

1. Download compose file Required

curl -LO https://raw.githubusercontent.com/shridarpatil/whatomate/main/docker/docker-compose.yml

2. Download config Required

curl -LO https://raw.githubusercontent.com/shridarpatil/whatomate/main/config.example.toml

3. Create config Required

cp config.example.toml config.toml

4. Run services Final step

docker compose up -d
